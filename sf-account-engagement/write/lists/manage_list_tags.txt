Tool: manage_list_tags

Description
Add or remove tags on a List.

Salesforce API
Account Engagement v5 REST API.

Endpoint
POST https://{pi-domain}/api/v5/objects/lists/:id/do/addTag
POST https://{pi-domain}/api/v5/objects/lists/:id/do/removeTag

Headers
Authorization: Bearer <salesforce-access-token>
Pardot-Business-Unit-Id: <buId>
Content-Type: application/json

Body
{ "tag": "MQL" }

Input (parameters)
id (number, required).
operation (string, required) - "add" | "remove".
tag (string, required).

Result
success (boolean), listId (number), tag (string), operation (string).

Required scopes
- `pardot_api`.

Idempotency & limits
- Idempotent.

Errors
- 59 not_found.

Example request

manage_list_tags({ "id": 301, "operation": "add", "tag": "MQL" })

Example result
{ "success": true, "listId": 301, "tag": "MQL", "operation": "add" }
