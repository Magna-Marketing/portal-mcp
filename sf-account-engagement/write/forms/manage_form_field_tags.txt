Tool: manage_form_field_tags

Description
Add or remove tags on a Form Field.

Salesforce API
Account Engagement v5 REST API.

Endpoint
POST https://{pi-domain}/api/v5/objects/form-fields/:id/do/addTag
POST https://{pi-domain}/api/v5/objects/form-fields/:id/do/removeTag

Headers
Authorization: Bearer <salesforce-access-token>
Pardot-Business-Unit-Id: <buId>
Content-Type: application/json

Body
{ "tag": "Required" }

Input (parameters)
id (number, required).
operation (string, required) - "add" | "remove".
tag (string, required).

Result
success (boolean), formFieldId (number), tag (string), operation (string).

Required scopes
- `pardot_api`.

Idempotency & limits
- Idempotent.

Errors
- 59 not_found.

Example request

manage_form_field_tags({ "id": 52, "operation": "add", "tag": "Required" })

Example result
{ "success": true, "formFieldId": 52, "tag": "Required", "operation": "add" }
