Tool: manage_opportunity_tags

Description
Add or remove tags on an Opportunity. Note that Pardot opportunities
are read-only mirrors of SFDC; only tags can be written from the
Account Engagement side.

Salesforce API
Account Engagement v5 REST API.

Endpoint
POST https://{pi-domain}/api/v5/objects/opportunities/:id/do/addTag
POST https://{pi-domain}/api/v5/objects/opportunities/:id/do/removeTag

Headers
Authorization: Bearer <salesforce-access-token>
Pardot-Business-Unit-Id: <buId>
Content-Type: application/json

Body
{ "tag": "Renewal" }

Input (parameters)
id (number, required) - Pardot opportunity id (not the SFDC id).
operation (string, required) - "add" | "remove".
tag (string, required).

Result
success (boolean), opportunityId (number), tag (string),
operation (string).

Required scopes
- `pardot_api`.

Idempotency & limits
- Idempotent.

Errors
- 59 not_found.

Example request

manage_opportunity_tags({ "id": 1101, "operation": "add", "tag": "Renewal" })

Example result
{ "success": true, "opportunityId": 1101, "tag": "Renewal", "operation": "add" }
