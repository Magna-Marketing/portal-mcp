Tool: get_section

Description
Returns a single section by GID. Sections are column/section containers within a project (e.g. "To Do", "In Progress"). Related: Get Sections for Project, Update Section, Get Tasks from Section.

Endpoint
GET https://app.asana.com/api/1.0/sections/{section_gid}

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
None (GET request).

Input (parameters)
section_gid (string, required) - Section GID. URI path parameter.
opt_fields (string, optional) - Comma-separated list of fields to return.

Result
Returns object with data (section object). Fields include gid, resource_type, name, created_at, project, project.name.

Example result

{
  "data": {
    "gid": "7778889990001112",
    "resource_type": "section",
    "name": "In Progress",
    "created_at": "2025-01-10T08:00:00.000Z",
    "project": { "gid": "1112223334445556", "name": "Launch Campaign", "resource_type": "project" }
  }
}
