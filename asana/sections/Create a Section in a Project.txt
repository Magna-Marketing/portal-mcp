Tool: create_section_for_project

Description
Creates a new section in a project. Body includes name (required). Section is appended to the end of the project. Related: Get Sections for Project, Get a Section, Insert Section (move/insert), Add Task to Section.

Endpoint
POST https://app.asana.com/api/1.0/projects/{project_gid}/sections

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
JSON. Required.
name (string, required) - Section name (e.g. "In Review").

Input (parameters)
project_gid (string, required) - Project GID. URI path parameter.

Result
Returns object with data (created section object).

Example request body

{
  "name": "In Review"
}

Example result

{
  "data": {
    "gid": "7778889990001114",
    "resource_type": "section",
    "name": "In Review",
    "created_at": "2025-03-05T12:00:00.000Z",
    "project": { "gid": "1112223334445556", "name": "Launch Campaign", "resource_type": "project" }
  }
}
