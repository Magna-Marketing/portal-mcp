Tool: create_status_for_object

Description
Creates a status update for a parent object (project or goal). Body includes parent (GID), title, optional text/html_text, status_type (on_track, at_risk, off_track, on_hold, complete). Related: Get Status Updates from an Object, Get a Status Update, Delete Status Update.

Endpoint
POST https://app.asana.com/api/1.0/status_updates

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
JSON. Required.
parent (string, required) - Parent object GID (project or goal).
title (string, optional) - Title of the update.
text (string, optional) - Plain text body.
html_text (string, optional) - HTML body.
status_type (string, optional) - "on_track", "at_risk", "off_track", "on_hold", "complete".

Input (parameters)
None (all in body).

Result
Returns object with data (created status_update object).

Example request body

{
  "parent": "1112223334445556",
  "title": "Weekly Update",
  "text": "All milestones on track.",
  "status_type": "on_track"
}

Example result

{
  "data": {
    "gid": "12345678",
    "resource_type": "status_update",
    "title": "Weekly Update",
    "text": "All milestones on track.",
    "status_type": "on_track",
    "created_at": "2025-03-05T12:00:00.000Z",
    "author": { "gid": "1210038679514467", "name": "Jane Doe", "resource_type": "user" },
    "parent": { "gid": "1112223334445556", "resource_type": "project", "name": "Launch Campaign" }
  }
}
