Tool: get_portfolio

Description
Returns a single portfolio by GID. Use for full portfolio details including optional fields when requested via opt_fields. Related: Get Multiple Portfolios (list), Get Portfolio Items, Add Members to Portfolio, Get Portfolio Memberships.

Endpoint
GET https://app.asana.com/api/1.0/portfolios/{portfolio_gid}

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
None (GET request).

Input (parameters)
portfolio_gid (string, required) - Portfolio GID. URI path parameter.
opt_fields (string, optional) - Comma-separated list of fields to return.
opt_expand (string, optional) - Comma-separated list of relation names to expand.

Result
Returns object with data (single portfolio object). Fields include gid, resource_type, name, archived, color, created_at, created_by, custom_field_settings, due_on, members, owner, start_on, workspace.
