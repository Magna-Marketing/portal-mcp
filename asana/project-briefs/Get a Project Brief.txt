Tool: get_project_brief

Description
Returns the project brief for a project. A project brief holds high-level context (title, body, content). Related: Update Project Brief, Create Project Brief.

Endpoint
GET https://app.asana.com/api/1.0/projects/{project_gid}/project_brief

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
None (GET request).

Input (parameters)
project_gid (string, required) - Project GID. URI path parameter.
opt_fields (string, optional) - Comma-separated list of fields to return.

Result
Returns object with data (project_brief object or null if none). Fields include gid, resource_type, title, html_text, status.

Example result

{
  "data": {
    "gid": "12345678",
    "resource_type": "project_brief",
    "title": "Q1 Launch Overview",
    "html_text": "<body>Goals and scope...</body>",
    "status": "complete"
  }
}
