Tool: get_dependencies_for_task

Description
Returns tasks that this task depends on (blocking tasks). Dependencies indicate "this task cannot be completed until these are done." Related: Set Dependencies for a Task, Unlink Dependencies, Get Dependents from a Task.

Endpoint
GET https://app.asana.com/api/1.0/tasks/{task_gid}/dependencies

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
None (GET request).

Input (parameters)
task_gid (string, required) - Task GID. URI path parameter.
opt_fields (string, optional) - Comma-separated list of fields to return on each dependency task.
limit (number, optional) - Results per page.
offset (string, optional) - Pagination offset.

Result
Returns object with data (array of task objects that are dependencies of the given task).

Example result

{
  "data": [
    {
      "gid": "1213485594364559",
      "resource_type": "task",
      "name": "Requirements doc",
      "completed": true,
      "created_at": "2026-03-01T10:00:00.000Z"
    }
  ]
}
