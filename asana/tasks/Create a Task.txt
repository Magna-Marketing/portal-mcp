Tool: create_task

Description
Creates a new task. Body must include workspace GID; name is typically required. Optional: notes, completed, due_on, assignee, projects (comma-separated), parent, start_on, resource_subtype. Related: Get a Task, Update Task, Create Subtask, Add Project to Task.

Endpoint
POST https://app.asana.com/api/1.0/tasks

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
JSON. Required.
workspace (string, required) - Workspace GID.
name (string, optional) - Task name (required for visible task in UI).
notes (string, optional) - Notes/description.
completed (boolean, optional) - Default false.
due_on (string, optional) - Due date YYYY-MM-DD.
assignee (string, optional) - User GID to assign.
projects (string, optional) - Comma-separated project GIDs to add task to.
parent (string, optional) - Parent task GID (creates subtask).
start_on (string, optional) - Start date YYYY-MM-DD.
resource_subtype (string, optional) - default_task or milestone.

Input (parameters)
None (all in body).

Result
Returns object with data (created task object).

Example request body

{
  "workspace": "9876543210987654",
  "name": "Design review",
  "notes": "Review design specs.",
  "due_on": "2026-06-30",
  "projects": "1112223334445556"
}

Example result

{
  "data": {
    "gid": "1213485594364562",
    "resource_type": "task",
    "name": "Design review",
    "completed": false,
    "due_on": "2026-06-30",
    "created_at": "2026-03-05T12:00:00.000Z",
    "workspace": { "gid": "9876543210987654", "name": "My Workspace", "resource_type": "workspace" },
    "projects": [{ "gid": "1112223334445556", "name": "Launch Campaign", "resource_type": "project" }]
  }
}
