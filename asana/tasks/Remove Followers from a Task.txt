Tool: remove_follower_for_task

Description
Removes a follower from a task. Body includes follower (user GID). Related: Add Followers to a Task, Get a Task.

Endpoint
POST https://app.asana.com/api/1.0/tasks/{task_gid}/removeFollowers

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
JSON. Required.
followers (string, required) - Comma-separated list of user GIDs to remove as followers.

Input (parameters)
task_gid (string, required) - Task GID. URI path parameter.

Result
Returns object with data (task object with updated followers list).

Example request body

{
  "followers": "1211007699959733"
}

Example result

{
  "data": {
    "gid": "1213485594364562",
    "resource_type": "task",
    "name": "Design review",
    "followers": [
      { "gid": "1210038679514467", "name": "Jane Doe", "resource_type": "user" }
    ]
  }
}
