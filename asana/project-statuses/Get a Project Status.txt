Tool: get_project_status

Description
Returns a single project status update by GID. Project statuses are progress updates (text, color, title) on a project. Related: Get Statuses from a Project, Create Project Status, Delete Project Status.

Endpoint
GET https://app.asana.com/api/1.0/project_statuses/{project_status_gid}

Headers (all Asana API calls)
Authorization: Bearer <access_token> (Personal Access Token or OAuth).
Content-Type: application/json

Body
None (GET request).

Input (parameters)
project_status_gid (string, required) - Project status GID. URI path parameter.
opt_fields (string, optional) - Comma-separated list of fields to return.

Result
Returns object with data (project_status object). Fields include gid, resource_type, title, text, color, created_at, created_by, author, modified_at, status_type (e.g. "on_track", "at_risk", "off_track"), project.

  }
}
