Tool: list_member_badges

Description
List `LoyaltyProgramMemberBadge` rows — the badges a member has been
awarded. Returns badge identity, awarded date, and (optionally) the
TransactionJournal that triggered the award.

Salesforce API
REST API (Data), v66.0. SOQL on `LoyaltyProgramMemberBadge`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, MemberId, Member.MembershipNumber, BadgeId, Badge.Name, Badge.BadgeType, Badge.ImageUrl, AwardedDate, ExpiryDate, RelatedTransactionJournalId, CreatedDate FROM LoyaltyProgramMemberBadge

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
memberId (string, optional).
membershipNumber (string, optional).
programName (string, optional) - Filter via Member.Program.Name.
badgeId (string, optional).
includeExpired (boolean, optional, default false).
limit (number, optional, default 200).

Result
totalSize (number).
memberBadges (array) - { id, memberId, membershipNumber, badgeId, badgeName, badgeType, imageUrl, awardedDate, expiryDate, relatedTransactionJournalId, createdDate }.

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_member_badges({ "memberId": "0lMQg00000abcDE2" })
