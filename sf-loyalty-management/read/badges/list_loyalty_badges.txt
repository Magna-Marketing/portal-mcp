Tool: list_loyalty_badges

Description
List `LoyaltyProgramBadge` records ‚Äî badges that members earn (e.g.
"First Purchase", "Frequent Flyer", "Top 10% Spender"). Includes image,
description, criteria, and validity.

Salesforce API
REST API (Data), v66.0. SOQL on `LoyaltyProgramBadge`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, Name, Description, ProgramId, Program.Name, BadgeType, ImageUrl, IsActive, EffectiveStartDate, EffectiveEndDate, MaxIssuanceCount, IssuedCount, CreatedDate FROM LoyaltyProgramBadge

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
programName (string, optional).
programId (string, optional).
badgeType (string, optional) - "Achievement" | "Milestone" | "Recognition" | etc.
isActive (boolean, optional).
namePattern (string, optional).
limit (number, optional, default 200).

Result
totalSize (number).
badges (array) - { id, name, description, programId, programName, badgeType, imageUrl, isActive, effectiveStartDate, effectiveEndDate, maxIssuanceCount, issuedCount, createdDate }.

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_loyalty_badges({ "programName": "MTG_Rewards", "isActive": true })
