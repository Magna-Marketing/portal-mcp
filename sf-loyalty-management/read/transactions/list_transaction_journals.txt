Tool: list_transaction_journals

Description
List `TransactionJournal` rows. Each journal is one accrual, redemption,
adjustment, or manual entry against a member, processed by the loyalty
rule engine. Filter by member, program, journal type/subtype, status,
date range. Use this for reconciliation and audits.

Salesforce API
REST API (Data), v66.0. SOQL on `TransactionJournal`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, JournalDate, JournalTypeId, JournalType.Name, JournalSubTypeId, JournalSubType.Name, Status, ProgramId, Program.Name, MemberId, Member.MembershipNumber, ActivityDate, EventType, ChannelId, ExternalTransactionNumber, TransactionAmount, CurrencyIsoCode, ProductId, ReferenceObjectName, ReferenceObjectId, CreatedDate FROM TransactionJournal

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
programName (string, optional).
memberId (string, optional).
membershipNumber (string, optional).
journalType (string|string[], optional) - "Accrual" | "Redemption" | "Adjustment" | "Manual" | "Burn" | "Cancel" | etc.
status (string|string[], optional) - "Pending" | "Processing" | "Processed" | "Cancelled" | "Failed".
sinceDays (number, optional, default 30).
externalTransactionNumber (string, optional) - Idempotency key check.
limit (number, optional, default 200, max 50000).
sortBy (string, optional, default "JournalDate").

Result
totalSize (number).
journals (array) - { id, journalDate, journalTypeId, journalTypeName, journalSubTypeId, journalSubTypeName, status, programId, programName, memberId, membershipNumber, activityDate, eventType, channelId, externalTransactionNumber, transactionAmount, currencyIsoCode, productId, referenceObjectName, referenceObjectId, createdDate }.
hasMore (boolean), nextRecordsUrl (string, optional).

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_transaction_journals({ "memberId": "0lMQg00000abcDE2", "journalType": ["Accrual","Redemption"], "sinceDays": 30 })
