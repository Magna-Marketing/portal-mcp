Tool: list_partner_promotions

Description
List partner-driven promotions (`LoyaltyPgmPartnerPromotion`) — the
join between a `LoyaltyProgramPartner` and a `Promotion`. Tells you
which promotions are co-funded or partner-exclusive.

Salesforce API
REST API (Data), v66.0. SOQL on `LoyaltyPgmPartnerPromotion`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, LoyaltyProgramPartnerId, LoyaltyProgramPartner.Name, PromotionId, Promotion.Name, IsActive, EffectiveStartDate, EffectiveEndDate, CreatedDate FROM LoyaltyPgmPartnerPromotion

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
partnerId (string, optional).
partnerName (string, optional).
promotionId (string, optional).
isActive (boolean, optional).
limit (number, optional, default 200).

Result
totalSize (number).
partnerPromotions (array) - { id, partnerId, partnerName, promotionId, promotionName, isActive, effectiveStartDate, effectiveEndDate, createdDate }.

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_partner_promotions({ "partnerName": "MTG_Hotel_Partner" })
