Tool: list_partner_products

Description
List `LoyaltyPartnerProduct` records — products offered by a partner
that earn / redeem within the loyalty program. Useful for partner-
catalogue UIs and rule scoping.

Salesforce API
REST API (Data), v66.0. SOQL on `LoyaltyPartnerProduct`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, Name, PartnerId, Partner.Name, PartnerProgramId, ProductId, Product.Name, ProductCategory, EarnRate, RedemptionRate, IsActive, EffectiveStartDate, EffectiveEndDate, CreatedDate FROM LoyaltyPartnerProduct

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
partnerId (string, optional).
programName (string, optional).
productCategory (string, optional).
isActive (boolean, optional).
limit (number, optional, default 200).

Result
totalSize (number).
partnerProducts (array) - { id, name, partnerId, partnerName, partnerProgramId, productId, productName, productCategory, earnRate, redemptionRate, isActive, effectiveStartDate, effectiveEndDate, createdDate }.

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_partner_products({ "partnerId": "0PpQg0000004ZyAA", "isActive": true })
