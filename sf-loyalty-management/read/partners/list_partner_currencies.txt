Tool: list_partner_currencies

Description
List `LoyaltyPgmPartnerCurrency` records — partner-specific currencies
mapped to one of the program's currencies (e.g. partner "Coins" -> 1
program "Point"). Used during partner-driven accruals.

Salesforce API
REST API (Data), v66.0. SOQL on `LoyaltyPgmPartnerCurrency`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, Name, LoyaltyProgramPartnerId, LoyaltyProgramPartner.Name, ProgramCurrencyId, ProgramCurrency.Name, ConversionRate, IsActive, EffectiveStartDate, EffectiveEndDate FROM LoyaltyPgmPartnerCurrency

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
partnerId (string, optional).
partnerName (string, optional).
programCurrencyName (string, optional).
isActive (boolean, optional).

Result
totalSize (number).
partnerCurrencies (array) - { id, name, partnerId, partnerName, programCurrencyId, programCurrencyName, conversionRate, isActive, effectiveStartDate, effectiveEndDate }.

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_partner_currencies({ "partnerName": "MTG_Hotel_Partner" })
