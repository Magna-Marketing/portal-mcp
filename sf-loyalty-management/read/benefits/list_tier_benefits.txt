Tool: list_tier_benefits

Description
List `LoyaltyTierBenefit` records — the join between a `LoyaltyTier`
and a `Benefit`. Tells you which benefits each tier grants.

Salesforce API
REST API (Data), v66.0. SOQL on `LoyaltyTierBenefit`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, LoyaltyTierId, LoyaltyTier.Name, LoyaltyTier.LoyaltyTierGroupId, BenefitId, Benefit.Name, Benefit.BenefitType.Name, IsActive, EffectiveStartDate, EffectiveEndDate, CreatedDate FROM LoyaltyTierBenefit

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
tierId (string, optional).
tierName (string, optional).
benefitId (string, optional).
programName (string, optional) - Filter via LoyaltyTier.LoyaltyTierGroup.Program.Name.
isActive (boolean, optional, default true).
limit (number, optional, default 200).

Result
totalSize (number).
tierBenefits (array) - { id, tierId, tierName, tierGroupId, benefitId, benefitName, benefitTypeName, isActive, effectiveStartDate, effectiveEndDate, createdDate }.

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_tier_benefits({ "tierName": "Gold" })
