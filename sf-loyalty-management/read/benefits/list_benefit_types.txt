Tool: list_benefit_types

Description
List `BenefitType` records — the categories of benefits available in
the org (Discount, BonusPoints, FreeShipping, EarlyAccess, etc.). Each
type defines a set of parameter slots the underlying benefits fill.

Salesforce API
REST API (Data), v66.0. SOQL on `BenefitType`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, Name, Description, IsActive, CreatedDate FROM BenefitType

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
isActive (boolean, optional, default true).

Result
totalSize (number).
benefitTypes (array) - { id, name, description, isActive, createdDate }.

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_benefit_types({})
