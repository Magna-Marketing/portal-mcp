Tool: list_member_vouchers

Description
List a member's vouchers via the Loyalty Connect API: identity, code,
status (Issued | Reserved | Redeemed | Cancelled | Expired), face
value, expiration. Use to render the member's wallet or to validate
before `redeem_voucher`.

Salesforce API
Loyalty Connect API, v66.0.

Endpoint
GET /services/data/v66.0/connect/loyalty/programs/{programName}/members/{memberId}/vouchers?status=Issued&pageSize=50

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
programName (string, required).
memberId (string, required).
status (string|string[], optional) - "Issued" | "Reserved" | "Redeemed" | "Cancelled" | "Expired".
voucherDefinitionName (string, optional).
includeExpired (boolean, optional, default false).
pageSize (number, optional, default 50, max 200).
pageToken (string, optional).

Result
memberId (string).
vouchers (array) - Each:
  voucherId (string), voucherCode (string),
  voucherDefinitionId (string), voucherDefinitionName (string),
  status (string),
  faceValue (number, optional), discountPercent (number, optional),
  effectiveDate (string|null), expirationDate (string|null),
  redeemedAt (string|null), cancelledAt (string|null),
  partnerAccountId (string|null), partnerName (string|null),
  isTransferable (boolean).
hasNextPage (boolean), nextPageToken (string|null).

Required permissions
- `Loyalty Management` permission set.

Idempotency & limits
- Read-only.

Errors
- LOYALTY_MEMBER_NOT_FOUND, LOYALTY_PROGRAM_NOT_FOUND.

Example request

list_member_vouchers({ "programName": "MTG_Rewards", "memberId": "0lMQg00000abcDE2", "status": "Issued" })
