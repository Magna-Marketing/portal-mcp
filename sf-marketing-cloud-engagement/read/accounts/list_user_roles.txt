Tool: list_user_roles

Description
List Marketing Cloud roles (`Role`) — both standard (Marketing Cloud
Administrator, Content Creator, Channel Manager, etc.) and custom.

Marketing Cloud API
SOAP API (`Role.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP retrieve on Role.

Input (parameters)
isCustomOnly (boolean, optional, default false).
namePattern (string, optional).

Result
totalSize (number).
roles (array) - { id (RoleID), name, description, isPrivate (boolean), isPrivilegedUserRole, customerKey, dateCreated, dateModified }.

Required scopes
- `users_read`.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_user_roles({ "isCustomOnly": true })
