Tool: list_users

Description
List Marketing Cloud users (`AccountUser`). Each user has identity, role,
business unit access, last login, locked-out flag.

Marketing Cloud API
SOAP API (`AccountUser.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP RetrieveRequest on `AccountUser`.

Input (parameters)
isActive (boolean, optional, default true).
emailPattern (string, optional) - Glob/substring on Email.
namePattern (string, optional) - Glob on Name.
mid (number, optional) - Filter by associated MID.
limit (number, optional, default 200).

Result
totalSize (number).
users (array) - { id (UserID), accountUserId, name, email, userId (Username), isApiUser, isActive, isLocked, isAdmin, isSendingUser, defaultApplicationId, lastSuccessfulLogin, lastFailedLogin, mustChangePassword, timezone, locale, roles (array of { id, name }) }.

Required scopes
- `users_read`.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_users({ "isActive": true })
