Tool: list_automations

Description
List Automation Studio automations. Each automation row includes Id,
name, status (Building / Scheduled / Running / Paused / Stopped /
Error), schedule, last run, and step count.

Marketing Cloud API
REST API (`/automation/v1/automations`).

Endpoint
GET https://{tenant}.rest.marketingcloudapis.com/automation/v1/automations?$page=1&$pageSize=50

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
status (string|string[], optional) - "Building" | "Scheduled" | "Running" | "Paused" | "Stopped" | "Error" | "Awaiting Trigger".
namePattern (string, optional).
folderId (number, optional).
modifiedSince (string, optional).
page (number, optional, default 1).
pageSize (number, optional, default 50, max 200).

Result
count (number).
items (array) - { id, key, name, description, statusId, status, scheduledTime, lastRunTime, lastRunStatus, scheduledStatus, categoryId, createdDate, modifiedDate, stepCount }.
hasMore (boolean), page (number).

Required scopes
- `automations_read`.

Idempotency & limits
- Read-only.

Errors
- 401 unauthorized.

Example request

list_automations({ "status": "Scheduled" })
