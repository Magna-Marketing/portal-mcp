Tool: list_opens

Description
List `OpenEvent` rows — every recorded open of an email by a
subscriber. Used for engagement reporting.

Marketing Cloud API
SOAP API (`OpenEvent.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP retrieve on OpenEvent with EventDate filter.

Input (parameters)
sinceDays (number, optional, default 1).
sendId (number, optional).
subscriberKey (string, optional).
limit (number, optional, default 1000).

Result
totalSize (number).
opens (array) - { sendId, subscriberKey, subscriberId, eventDate, triggererSendDefinitionObjectId, isUnique, browser, region, city, country }.
hasMore (boolean), continueRequestId (string|null).

Required scopes
- `tracking_events_read`.

Idempotency & limits
- Read-only.

Errors
- 401 unauthorized.

Example request

list_opens({ "sendId": 1234567, "sinceDays": 7 })
