Tool: list_clicks

Description
List `ClickEvent` rows — clicks tracked on links inside sent emails.

Marketing Cloud API
SOAP API (`ClickEvent.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP retrieve on ClickEvent.

Input (parameters)
sinceDays (number, optional, default 1).
sendId (number, optional).
subscriberKey (string, optional).
url (string, optional) - Filter by clicked URL.
limit (number, optional, default 1000).

Result
totalSize (number).
clicks (array) - { sendId, subscriberKey, subscriberId, eventDate, url, isUnique, urlId, browser, region, city, country }.
hasMore (boolean), continueRequestId (string|null).

Required scopes
- `tracking_events_read`.

Idempotency & limits
- Read-only.

Errors
- 401 unauthorized.

Example request

list_clicks({ "sendId": 1234567 })
