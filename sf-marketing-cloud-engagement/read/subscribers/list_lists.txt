Tool: list_lists

Description
List subscriber lists (`List`) — both system lists ("All Subscribers")
and custom publication / suppression lists. Use when you need to find
the right list Id for `add_subscribers_to_list` /
`remove_subscribers_from_list`.

Marketing Cloud API
SOAP API (`List.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP retrieve on List.

Input (parameters)
type (string, optional) - "Public" | "Private" | "Salesforce" | "GlobalUnsubscribe" | "Master".
namePattern (string, optional).
folderId (number, optional).
limit (number, optional, default 200).

Result
totalSize (number).
lists (array) - { id (ListID), customerKey, listName, description, type, listClassification ("ExactTargetList" | "PublicationList" | "SuppressionList"), category (folder Id), subscriberCount (when retrievable), dateCreated, dateModified }.

Required scopes
- `list_and_subscribers_read`.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_lists({ "type": "Public" })
