Tool: get_contact_exit_status

Description
Check the status of a contact-exit operation queued by
`remove_contact_from_journey`.

Marketing Cloud API
REST API (`/interaction/v1/interactions/contactexit/status`).

Endpoint
POST https://{tenant}.rest.marketingcloudapis.com/interaction/v1/interactions/contactexit/status

Headers
Authorization: Bearer <access_token>
Content-Type: application/json

Body
{ "OperationId": "abc-456-def" }

Input (parameters)
operationId (string, required).

Result
operation (object) - { operationId, status ("Pending"|"InProgress"|"Complete"|"Failed"), startedAt, completedAt, totalContacts, processedContacts, failedContacts, errors (array) }.

Required scopes
- `journeys_read`.

Idempotency & limits
- Read-only.

Errors
- 30002 operation not found.

Example request

get_contact_exit_status({ "operationId": "abc-456-def" })
