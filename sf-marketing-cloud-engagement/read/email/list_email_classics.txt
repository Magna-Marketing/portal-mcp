Tool: list_email_classics

Description
List classic Email Studio emails (`Email`). For Content Builder
HTML / template assets use `list_assets` with `assetTypeId=208`
(htmlemail) instead.

Marketing Cloud API
SOAP API (`Email.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP retrieve on Email.

Input (parameters)
folderId (number, optional) - Category Id.
namePattern (string, optional).
modifiedSince (string, optional).
emailType (string, optional) - "Standard" | "Triggered" | "Welcome".
limit (number, optional, default 200).

Result
totalSize (number).
emails (array) - { id, customerKey, name, subject, fromName, fromEmail, contentAreas (count), htmlBody (excerpt), textBody (excerpt), categoryId, isHTMLPaste, emailType, dateCreated, dateModified }.

Required scopes
- `email_read`.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_email_classics({ "folderId": 12345 })
