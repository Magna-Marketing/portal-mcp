Tool: list_delivery_profiles

Description
List `DeliveryProfile` records (delivery settings, Sender
Authentication Package, BCC behavior, dynamic email).

Marketing Cloud API
SOAP API (`DeliveryProfile.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP retrieve on DeliveryProfile.

Input (parameters)
namePattern (string, optional).
limit (number, optional, default 200).

Result
totalSize (number).
deliveryProfiles (array) - { id, customerKey, name, description, dynamicEmailSubject, sourceAddressType, privateIPAddress, autoBccEmail, headerSalutationSource, headerContentArea, footerSalutationSource, footerContentArea, domainType, dateCreated, dateModified }.

Required scopes
- `email_read`.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_delivery_profiles({})
