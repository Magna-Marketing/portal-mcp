Tool: list_send_classifications

Description
List `SendClassification` records. A Send Classification bundles a
Sender Profile and Delivery Profile and is required on most send
endpoints (Triggered Sends, Automations, Journey email activities).

Marketing Cloud API
SOAP API (`SendClassification.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP retrieve on SendClassification.

Input (parameters)
classificationType (string, optional) - "Default Commercial" | "Default Transactional" | etc.
namePattern (string, optional).
limit (number, optional, default 200).

Result
totalSize (number).
sendClassifications (array) - { id, customerKey, name, description, sendClassificationType, senderProfileCustomerKey, deliveryProfileCustomerKey, archiveEmail, honorPublicationListOptOutsForTransactionalSends, sendPriority, dateCreated, dateModified }.

Required scopes
- `email_read`.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_send_classifications({})
