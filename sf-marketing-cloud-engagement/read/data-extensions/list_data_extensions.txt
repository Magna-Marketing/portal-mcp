Tool: list_data_extensions

Description
List Data Extensions (`DataExtension`) in the current Business Unit.
Each entry includes identity, External Key (`CustomerKey`), folder,
sendable + testable flags, row count summary.

Marketing Cloud API
SOAP API (`DataExtension.Retrieve`).

Endpoint
POST https://{tenant}.soap.marketingcloudapis.com/Service.asmx

Headers
Authorization: Bearer <access_token>
Content-Type: text/xml; charset=utf-8

Body
SOAP RetrieveRequest on `DataExtension`.

Input (parameters)
folderId (number, optional).
namePattern (string, optional).
isSendable (boolean, optional).
isTestable (boolean, optional).
modifiedSince (string, optional) - ISO timestamp.
limit (number, optional, default 200).

Result
totalSize (number).
dataExtensions (array) - { id, customerKey (External Key), name, description, isSendable, isTestable, sendableDataExtensionField, sendableSubscriberField, categoryId, dataRetentionPeriodLength, dataRetentionPeriod, isDataRetentionEnabled, rowDeletionType, dateCreated, dateModified, rowCount (number, when retrievable) }.

Required scopes
- `data_extensions_read`.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_data_extensions({ "folderId": 12345, "isSendable": true })
