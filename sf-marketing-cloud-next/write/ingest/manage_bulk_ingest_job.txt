Tool: manage_bulk_ingest_job

Description
Create, close (mark as `UploadComplete`), abort, or delete a Data
Cloud Bulk Ingestion job. The lifecycle:

1. `manage_bulk_ingest_job operation="create"` → returns `jobId` in
   state `Open`.
2. `upload_bulk_ingest_data` → upload one or more CSV chunks.
3. `manage_bulk_ingest_job operation="close"` → moves job to
   `UploadComplete`; the platform processes asynchronously into
   `JobComplete`.
4. Poll with `read/ingest/get_ingest_job` until state is terminal.
5. (Optional) `manage_bulk_ingest_job operation="abort"` while
   running, or `operation="delete"` after completion.

Salesforce API
Data Cloud Bulk Ingestion API.

Endpoint
POST   {cdpInstanceUrl}/api/v1/ingest/jobs                      (operation="create")
PATCH  {cdpInstanceUrl}/api/v1/ingest/jobs/:jobId               (operation="close" / "abort")
DELETE {cdpInstanceUrl}/api/v1/ingest/jobs/:jobId               (operation="delete")

Headers
Authorization: Bearer <data_cloud_token>
Content-Type: application/json
Accept: application/json

Body (create)
{
  "object": "Order",
  "sourceName": "ecommerce",
  "operation": "upsert",
  "contentType": "CSV",
  "columnDelimiter": "COMMA",
  "lineEnding": "LF"
}

Body (close)
{ "state": "UploadComplete" }

Body (abort)
{ "state": "Aborted" }

Input (parameters)
operation (string, required) - "create" | "close" | "abort" | "delete".
jobId (string, required for close / abort / delete).
object (string, required for create) - Source object api name.
sourceName (string, required for create) - Data Stream source name.
ingestOperation (string, optional, default "upsert") - "upsert" |
  "insert" | "delete".
contentType (string, optional, default "CSV") - "CSV" | "JSON".
columnDelimiter (string, optional, default "COMMA") - "COMMA" |
  "PIPE" | "SEMICOLON" | "TAB".
lineEnding (string, optional, default "LF") - "LF" | "CRLF".
externalIdField (string, optional) - For upsert: the field used
  as the matching key.

Result
job (object) - Created / updated job (see `read/ingest/get_ingest_job`).
deleted (boolean, when operation="delete").

Required scopes
- `cdp_ingest_api`.

Idempotency & limits
- Not idempotent on create.
- 150GB per job; max 100 active jobs per tenant.

Errors
- 400 INVALID_STATE_TRANSITION (close while in `Aborted`),
  404 NOT_FOUND, 409 JOB_NOT_OPEN (cannot close a job already
  closed), 401 invalid_token.

Example request

manage_bulk_ingest_job({
  "operation": "create",
  "object": "Order",
  "sourceName": "ecommerce",
  "ingestOperation": "upsert",
  "externalIdField": "Id__c"
})

Example result
{
  "job": {
    "id": "JOB_001", "sourceApiName": "ecommerce",
    "objectApiName": "Order", "operation": "upsert",
    "state": "Open", "contentType": "CSV"
  }
}
