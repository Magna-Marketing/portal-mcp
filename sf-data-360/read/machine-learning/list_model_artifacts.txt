Tool: list_model_artifacts

Description
List Model Artifacts — the underlying packaged ML model files
(weights, code, metadata) that Configured Models reference.

Salesforce API
Connect plane.

Endpoint
GET {instanceUrl}/services/data/v{ver}/ssot/machine-learning/model-artifacts

Headers
Authorization: Bearer <core_access_token>
Accept: application/json

Body
None.

Input (parameters)
runtimeFramework (string, optional) - "TensorFlow" | "PyTorch" |
  "ONNX" | "ScikitLearn" | "AmazonSageMaker" | "VertexAI" |
  "AzureML".
status (string, optional) - "Active" | "Archived" | "Failed".
limit (number, optional, default 100).

Result
modelArtifacts (array) - Each: { id, apiName, label, version,
  runtimeFramework, status, sourceUrl, sizeBytes, createdDate,
  lastModifiedDate }.
totalSize (number).

Required scopes
- `api`.

Idempotency & limits
- Read-only.

Errors
- 401 INVALID_SESSION_ID.

Example request

list_model_artifacts({ "runtimeFramework": "TensorFlow" })

Example result
{
  "modelArtifacts": [
    { "id": "MA_001", "apiName": "ChurnArtifact_v12",
      "version": "1.2.0", "runtimeFramework": "TensorFlow",
      "status": "Active", "sizeBytes": 412800 }
  ]
}
