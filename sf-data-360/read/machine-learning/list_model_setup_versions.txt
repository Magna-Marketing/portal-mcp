Tool: list_model_setup_versions

Description
List the versions of a Model Setup — Einstein Studio / Model Builder
training configurations. Each version captures a snapshot of the
training config, dataset, and resulting metrics.

Salesforce API
Connect plane.

Endpoint
GET {instanceUrl}/services/data/v{ver}/ssot/machine-learning/model-setups/:modelSetupIdOrName/setup-versions

Headers
Authorization: Bearer <core_access_token>
Accept: application/json

Body
None.

Input (parameters)
modelSetupIdOrName (string, required).
status (string, optional) - "Active" | "Training" | "Failed" |
  "Archived".
limit (number, optional, default 100).

Result
versions (array) - Each: { id, apiName, version, status,
  modelArtifactId, trainingDatasetRef, hyperparameters (object),
  metrics (object), createdDate, lastModifiedDate }.
totalSize (number).

Required scopes
- `api`.

Idempotency & limits
- Read-only.

Errors
- 404 NOT_FOUND.

Example request

list_model_setup_versions({ "modelSetupIdOrName": "ChurnModel" })

Example result
{
  "versions": [
    { "id": "MSV_001", "version": "1.2.0", "status": "Active",
      "metrics": { "accuracy": 0.84, "f1": 0.79 } }
  ]
}
