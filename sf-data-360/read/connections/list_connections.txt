Tool: list_connections

Description
List Data 360 Connections — the source-system bindings (S3, SFTP,
Snowflake, Marketing Cloud, MC Engagement, Salesforce sObjects,
GA4, etc.) that Data Streams pull from. One row per configured
connection.

Salesforce API
Connect plane.

Endpoint
GET {instanceUrl}/services/data/v{ver}/ssot/connections

Headers
Authorization: Bearer <core_access_token>
Accept: application/json

Body
None.

Input (parameters)
connectorType (string, optional) - "S3" | "SFTP" | "Snowflake" |
  "MarketingCloud" | "MarketingCloudEngagement" | "Salesforce" |
  "GA4" | "GoogleBigQuery" | "AmazonRedshift" | "DatabricksDelta" |
  "GenericREST" | etc.
status (string, optional) - "Active" | "Disabled" | "Failed".
nameContains (string, optional).
limit (number, optional, default 100).
offset (number, optional).

Result
connections (array) - Each: { id, name, label, connectorType,
  description, status, isDefaultConnection, createdById, createdDate,
  lastModifiedById, lastModifiedDate, additionalProperties (object) }.
nextPageUrl (string|null), totalSize (number).

Required scopes
- `api`.

Idempotency & limits
- Read-only.

Errors
- 401 INVALID_SESSION_ID, 403 INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_connections({ "connectorType": "MarketingCloudEngagement" })

Example result
{
  "connections": [
    { "id": "0Z65800000000ABCAA", "name": "MTG_MCE",
      "label": "MTG Marketing Cloud Engagement",
      "connectorType": "MarketingCloudEngagement", "status": "Active" }
  ],
  "totalSize": 1
}
