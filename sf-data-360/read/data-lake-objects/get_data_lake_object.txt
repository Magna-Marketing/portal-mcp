Tool: get_data_lake_object

Description
Read a single DLO by id or developer name with full field
metadata.

Salesforce API
Connect plane.

Endpoint
GET {instanceUrl}/services/data/v{ver}/ssot/data-lake-objects/:recordIdOrDeveloperName

Headers
Authorization: Bearer <core_access_token>
Accept: application/json

Body
None.

Input (parameters)
recordIdOrDeveloperName (string, required).

Result
dataLakeObject (object): id, developerName, label, category,
  sourceDataStreamId, fields, recordCount, createdDate,
  lastModifiedDate.

Required scopes
- `api`.

Idempotency & limits
- Read-only.

Errors
- 404 NOT_FOUND.

Example request

get_data_lake_object({ "recordIdOrDeveloperName": "mc_sends__dll" })

Example result
{
  "dataLakeObject": {
    "developerName": "mc_sends__dll", "category": "Engagement",
    "sourceDataStreamId": "0bcxx0000000001"
  }
}
