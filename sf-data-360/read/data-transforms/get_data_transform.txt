Tool: get_data_transform

Description
Read a single Data Transform by id or name with the full SQL
definition and source / target metadata.

Salesforce API
Connect plane.

Endpoint
GET {instanceUrl}/services/data/v{ver}/ssot/data-transforms/:dataTransformNameOrId

Headers
Authorization: Bearer <core_access_token>
Accept: application/json

Body
None.

Input (parameters)
dataTransformNameOrId (string, required).

Result
dataTransform (object): id, developerName, label, type, targetObject,
  sourceObjects, sql, schedule, status, lastRunStatus, lastRunDate,
  formula (object|null), createdDate, lastModifiedDate.

Required scopes
- `api`.

Idempotency & limits
- Read-only.

Errors
- 404 NOT_FOUND.

Example request

get_data_transform({ "dataTransformNameOrId": "build_unified_individual" })

Example result
{
  "dataTransform": {
    "developerName": "build_unified_individual",
    "type": "Batch",
    "sql": "SELECT * FROM Contact__dlm UNION ALL SELECT * FROM Lead__dlm"
  }
}
