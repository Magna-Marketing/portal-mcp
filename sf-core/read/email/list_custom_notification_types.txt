Tool: list_custom_notification_types

Description
List Custom Notification Types (`CustomNotificationType`). These are
the channels you can `publish_custom_notification` to (in-app + mobile).
Each type has its supported channels (Desktop / Mobile) and an icon.

Salesforce API
REST API (Data) + Tooling, v66.0. SOQL on `CustomNotificationType`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, DeveloperName, MasterLabel, Description, CustomNotifTypeName, Title, Body, IconName, NotificationChannel FROM CustomNotificationType

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).

Result
totalSize (number).
notificationTypes (array) - { id, developerName, masterLabel, description, supportedChannels (string[]), iconName, defaultTitle, defaultBody }.

Required permissions
- "API Enabled" + "Customize Application".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_custom_notification_types({})
