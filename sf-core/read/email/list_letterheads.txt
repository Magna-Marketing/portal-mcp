Tool: list_letterheads

Description
List Letterheads (`BrandTemplate`) used as a header/footer container
for HTML email templates.

Salesforce API
REST API (Data), v66.0. SOQL on `BrandTemplate`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, Name, DeveloperName, IsActive, Description FROM BrandTemplate

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
includeInactive (boolean, optional, default false).
namePattern (string, optional).

Result
totalSize (number).
letterheads (array) - { id, name, developerName, isActive, description }.

Required permissions
- "API Enabled".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_letterheads({ "includeInactive": false })
