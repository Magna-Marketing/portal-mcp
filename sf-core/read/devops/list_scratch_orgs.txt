Tool: list_scratch_orgs

Description
List scratch org definitions and active scratch orgs (`ScratchOrgInfo`).
Each row shows the configuration, status, expiration, and connected
edition. Scratch orgs are managed from a Dev Hub org.

Salesforce API
Tooling API, v66.0. `ScratchOrgInfo`.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT Id, OrgName, Status, ExpirationDate, Edition, Country, Description, Features, Namespace, ConnectedAppConsumerKey, ConnectedAppCallbackUrl, AdminEmail, Username, AuthCode, LoginUrl, SignupInstance, ScratchOrg, ScratchOrgInfoVersion, CreatedById, CreatedDate FROM ScratchOrgInfo

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
status (string, optional) - "New" | "InProgress" | "Active" | "Error" | "Deleted".
includeExpired (boolean, optional, default false).
limit (number, optional, default 100).

Result
totalSize (number).
scratchOrgs (array) - { id, orgName, status, expirationDate, edition, country, namespace, adminEmail, username, signupInstance, scratchOrg (Org Id), createdBy, createdDate }.

Required permissions
- Run from a Dev Hub org with "Manage Dev Hub" + "View All Data".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY, INVALID_ORG_TYPE if not Dev Hub.

Example request

list_scratch_orgs({ "status": "Active" })
