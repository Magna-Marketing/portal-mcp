Tool: list_apex_test_suites

Description
List `ApexTestSuite` records and their member test classes. Use to run
a curated subset of tests (`run_apex_tests` with `testSuiteNames`).

Salesforce API
Tooling API, v66.0. `ApexTestSuite` + `TestSuiteMembership`.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT Id, TestSuiteName FROM ApexTestSuite
GET /services/data/v66.0/tooling/query/?q=SELECT ApexTestSuiteId, ApexClassId, ApexClass.Name FROM TestSuiteMembership WHERE ApexTestSuiteId IN (...)

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
includeMembers (boolean, optional, default true).

Result
totalSize (number).
testSuites (array) - { id, testSuiteName, members (array, when includeMembers=true): [{ classId, className }] }.

Required permissions
- "View All Data" / "Author Apex".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_apex_test_suites({ "includeMembers": true })
