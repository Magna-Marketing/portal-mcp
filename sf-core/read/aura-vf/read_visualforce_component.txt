Tool: read_visualforce_component

Description
Read a single Visualforce component (`ApexComponent`) with body and
controller binding.

Salesforce API
Tooling API, v66.0.

Endpoint
GET /services/data/v66.0/tooling/sobjects/ApexComponent/{id}

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
name (string, required) - VF component Name.

Result
component (object) - { id, name, masterLabel, apiVersion, controllerType, namespacePrefix, manageableState, description, markup (string), createdDate, lastModifiedDate }.

Required permissions
- "Customize Application" / "Author Apex".

Idempotency & limits
- Read-only.

Errors
- NOT_FOUND.

Example request

read_visualforce_component({ "name": "AccountHeader" })
