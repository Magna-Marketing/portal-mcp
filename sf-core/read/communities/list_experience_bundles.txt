Tool: list_experience_bundles

Description
List `ExperienceBundle` records — the static-resource backed source for
LWR / Aura Experience sites. Returns bundle identity, deploy status,
and the routes inside.

Salesforce API
Tooling API, v66.0.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT Id, DeveloperName, MasterLabel, NamespacePrefix, Status FROM ExperienceBundle

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
status (string, optional) - "Active" | "Inactive" | "DraftPublished".
includeRoutes (boolean, optional, default false).

Result
totalSize (number).
experienceBundles (array) - { id, developerName, masterLabel, namespacePrefix, status, routes (array, optional): [{ pathPrefix, pageName }] }.

Required permissions
- "Create and Set Up Experiences" / "Modify All Data".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_experience_bundles({})
