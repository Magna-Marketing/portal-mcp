Tool: list_content_assets

Description
List `ContentAsset` records (Files used inside Setup, branding, themes,
help content). They are referenced by name (e.g. `$ContentAsset.MyAsset`
in Apex/Lightning).

Salesforce API
REST API (Data), v66.0. SOQL on `ContentAsset`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, DeveloperName, MasterLabel, IsVisibleByExternalUsers, NamespacePrefix, FileExtension, ContentDocumentId FROM ContentAsset

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
includeManaged (boolean, optional, default false).

Result
totalSize (number).
contentAssets (array) - { id, developerName, masterLabel, namespacePrefix, fileExtension, contentDocumentId, isVisibleByExternalUsers }.

Required permissions
- "Customize Application".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_content_assets({})
