Tool: list_content_distributions

Description
List `ContentDistribution` rows (public file links). Each row gives the
distribution Id, password protection, expiration, and the public URL
the recipient uses.

Salesforce API
REST API (Data), v66.0. SOQL on `ContentDistribution`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, ContentDocumentId, ContentVersionId, Name, DistributionPublicUrl, ExpiryDate, RelatedRecordId, PasswordRequired, ViewCount, CreatedDate, LastViewDate FROM ContentDistribution

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
contentDocumentId (string, optional).
relatedRecordId (string, optional).
includeExpired (boolean, optional, default false).
limit (number, optional, default 200).

Result
totalSize (number).
contentDistributions (array) - { id, contentDocumentId, contentVersionId, name, distributionPublicUrl, expiryDate, relatedRecordId, passwordRequired, viewCount, createdDate, lastViewDate }.

Required permissions
- "API Enabled" + file read.

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_content_distributions({ "contentDocumentId": "069Qg0000004ZyAA" })
