Tool: list_external_client_apps

Description
List External Client Apps (`ExternalClientApplication`), Salesforce's
successor to Connected Apps. Includes OAuth config, policies (IP, refresh,
session), and per-app usage stats.

Salesforce API
Tooling + Connect API, v66.0.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT Id, DeveloperName, MasterLabel, IsActive, IsClientCredentialsFlowEnabled, IsHybridAppFlowEnabled, IsPkceRequired FROM ExternalClientApplication
GET /services/data/v66.0/apps/oauth/usage    (cross-reference)

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
includeUsage (boolean, optional, default true).
includeOAuthSettings (boolean, optional, default true).

Result
totalSize (number).
externalClientApps (array) - { id, developerName, masterLabel, isActive, isClientCredentialsFlowEnabled, isHybridAppFlowEnabled, isPkceRequired, oauth (object|null), usage (object|null): { totalApiCalls, lastUsedDate, totalUsers } }.

Required permissions
- "Manage Connected Apps".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_external_client_apps({})
