Tool: list_connected_apps

Description
List Connected Apps in the org with OAuth scopes, callback URLs, and
policy settings. For modern apps prefer External Client Apps
(`list_external_client_apps`).

Salesforce API
Tooling API + Metadata API, v66.0. `ConnectedApplication`.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT Id, Name, NamespacePrefix, CreatedDate, OptionsAllowAdminApprovedUsersOnly FROM ConnectedApplication
POST /services/Soap/m/66.0   (readMetadata ConnectedApp for full XML)

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
includePolicies (boolean, optional, default true) - IP relaxation, IP allowlist, refresh token policy, mobile session timeout.
includeOAuthSettings (boolean, optional, default true) - Scopes, callback URL, consumer key.

Result
totalSize (number).
connectedApps (array) - Each: { id, name, namespacePrefix, label, contactEmail, description, infoUrl, iconUrl, isAdminApprovedUsersOnly, oauth (object|null): { scopes, callbackUrl, consumerKey, isPkceRequired, isSecretRequiredForRefreshToken, idTokenConfig }, policies (object|null): { ipRelaxation, ipAllowlist, refreshTokenPolicy, sessionTimeout, requireUsersToInstallApp } }.

Required permissions
- "Manage Connected Apps" / "View Setup and Configuration".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_connected_apps({ "includePolicies": true })
