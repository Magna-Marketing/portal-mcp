Tool: list_oauth_tokens

Description
List active OAuth tokens (`OauthToken`): which user, which app, when
issued, last used, and what scopes they hold. Use to audit / revoke
stale tokens (`revoke_oauth_token`).

Salesforce API
REST API (Data), v66.0. SOQL on `OauthToken`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, AppName, AppMenuItemId, UserId, User.Username, CreatedDate, LastUsedDate, Scopes, UseCount, DeleteToken FROM OauthToken

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
userId (string, optional).
appName (string, optional).
scopesContains (string, optional) - e.g. "refresh_token".
sinceLastUsedHours (number, optional) - Only tokens used in the last N hours.
limit (number, optional, default 500).

Result
totalSize (number).
tokens (array) - { id, appName, userId, username, createdDate, lastUsedDate, scopes (string[]), useCount }.

Required permissions
- "Manage Connected Apps" + "Manage Users".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_oauth_tokens({ "scopesContains": "refresh_token" })
