Tool: list_cors_origins

Description
List CORS allowlisted origins (`CorsWhitelistEntry`) — the domains
allowed to call the org's REST APIs from a browser.

Salesforce API
Tooling API, v66.0. `CorsWhitelistEntry`.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT Id, UrlPattern FROM CorsWhitelistEntry

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
None.

Result
totalSize (number).
corsOrigins (array) - { id, urlPattern }.

Required permissions
- "Modify All Data".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_cors_origins({})
