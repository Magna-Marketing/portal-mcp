Tool: list_custom_settings

Description
List Custom Settings (List or Hierarchy) in the org with their fields
and record counts. Custom Settings are sObjects with `customSetting=true`
in describe. Hierarchy custom settings have an implicit per-user /
per-profile / org-default record selection.

Salesforce API
REST API (Data) + Tooling, v66.0.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT QualifiedApiName, Label, PluralLabel, NamespacePrefix FROM EntityDefinition WHERE IsCustomSetting = true
GET /services/data/v66.0/sobjects/{name}/describe

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
type (string, optional) - "list" | "hierarchy".
includeFields (boolean, optional, default true).
includeRecordCounts (boolean, optional, default true).

Result
totalSize (number).
customSettings (array) - Each:
  qualifiedApiName, label, pluralLabel, namespacePrefix.
  type (string) - "list" | "hierarchy".
  visibility (string) - "Public" | "Protected".
  fields (array, when includeFields=true) - { apiName, label, type, length, defaultValue }.
  recordCount (number, when includeRecordCounts=true).

Required permissions
- "Customize Application".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_custom_settings({ "type": "hierarchy", "includeRecordCounts": true })
