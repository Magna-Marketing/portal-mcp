Tool: get_org_info

Description
Get the org's identity: organization name, edition, instance URL,
namespace prefix, default locale/language, fiscal year start,
sandbox flag, trial expiration, organization Id (15+18 char), and a
list of accepted API versions.

Salesforce API
REST API + Data, v66.0. SOQL on `Organization` + `/services/data`.

Endpoint
GET /services/data
GET /services/data/v66.0/query/?q=SELECT Id, Name, OrganizationType, IsSandbox, InstanceName, NamespacePrefix, LanguageLocaleKey, DefaultLocaleSidKey, FiscalYearStartMonth, UsesStartDateAsFiscalYearName, ComplianceBccEmail, PrimaryContact, Phone, FaxLine1, Street, City, State, PostalCode, Country, Latitude, Longitude, GeocodeAccuracy, TrialExpirationDate, MonthlyPageViewsUsed, MonthlyPageViewsEntitlement FROM Organization

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
None.

Result
organization (object):
  id (string), name, organizationType (e.g. "Developer Edition"|"Enterprise"|"Unlimited"|"Performance"|"Group"|"Professional"|"Personal"|"Force.com Free"),
  isSandbox (boolean), instanceName (string), namespacePrefix (string|null),
  languageLocaleKey, defaultLocaleSidKey, fiscalYearStartMonth, primaryContact, phone,
  trialExpirationDate (string|null), monthlyPageViewsUsed, monthlyPageViewsEntitlement.
apiVersions (array) - Each: { version, label, url }.
instanceUrl (string).

Required permissions
- "API Enabled".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

get_org_info({})
