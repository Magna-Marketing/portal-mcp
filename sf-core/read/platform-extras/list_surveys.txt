Tool: list_surveys

Description
List `Survey` records and their invitations / responses summary. Used
for NPS, CSAT, post-case surveys.

Salesforce API
REST API (Data), v66.0.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, Name, Description, OwnerId, Status, IsActive, CreatedDate FROM Survey
GET /services/data/v66.0/query/?q=SELECT Id, SurveyId, Name, NumberOfResponses, NumberOfInvitations FROM SurveyVersion WHERE SurveyId IN (...)

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
status (string, optional).
isActive (boolean, optional).
includeStats (boolean, optional, default true).

Result
totalSize (number).
surveys (array) - { id, name, description, status, isActive, ownerId, ownerName, latestVersion (object, when includeStats=true): { id, numberOfInvitations, numberOfResponses } }.

Required permissions
- "Manage Surveys" / "View All Data".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_surveys({ "isActive": true })
