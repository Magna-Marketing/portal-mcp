Tool: list_big_objects

Description
List Big Objects in the org (entities ending in `__b`). Big Objects
support billions of records and are queried with SOQL but cannot be
related to standard objects via lookups.

Salesforce API
Tooling API + REST API, v66.0.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT QualifiedApiName, Label, PluralLabel FROM EntityDefinition WHERE IsCustomBigObject = true
GET /services/data/v66.0/sobjects/{name}/describe

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
includeIndexFields (boolean, optional, default true) - Big Objects require an index; show its fields.

Result
totalSize (number).
bigObjects (array) - { qualifiedApiName, label, pluralLabel, indexFields (array, when includeIndexFields=true): [{ apiName, label, type, sortDirection }] }.

Required permissions
- "Customize Application".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_big_objects({})
