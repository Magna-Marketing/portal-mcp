Tool: list_apex_rest_endpoints

Description
List custom Apex REST endpoints (`@RestResource(urlMapping=...)`)
defined in Apex classes. Useful for understanding the org's bespoke
REST surface, accessible at `/services/apexrest/<urlMapping>`.

Salesforce API
Tooling API, v66.0. SOQL on `ApexClass` body parsed for the
`@RestResource` annotation.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT Id, Name, Body, ApiVersion, Status FROM ApexClass WHERE Body LIKE '%@RestResource%'

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
includeManaged (boolean, optional, default false).
namespacePrefix (string, optional).

Result
totalSize (number).
apexRestEndpoints (array) - { className, classId, namespacePrefix, urlMapping (string), httpMethods (string[]) - "GET" | "POST" | "PUT" | "PATCH" | "DELETE" found via @HttpGet/@HttpPost/etc., methodNames (object) - { GET, POST, PUT, PATCH, DELETE: methodName }, apiVersion, status }.

Required permissions
- "Author Apex" / "View All Data".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_apex_rest_endpoints({})
