Tool: list_report_subscriptions

Description
List report subscriptions (scheduled email deliveries of a report's
results to one or more users). Each subscription shows the report,
recipients, schedule, conditions for sending, and last run.

Salesforce API
Analytics REST API, v66.0. `/analytics/notifications`.

Endpoint
GET /services/data/v66.0/analytics/notifications?source=...

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
reportId (string, optional).
ownerId (string, optional).
includeInactive (boolean, optional, default false).

Result
subscriptions (array) - { id, name, source ("report"), reportId, recipients (array): [{ id, name, type ("user"|"group"|"role") }], schedule (object): { type ("hourly"|"daily"|"weekly"|"monthly"), startTime, frequency, daysOfWeek }, conditions (array): [{ aggregate, operator, value }], active (boolean), lastNotificationDate, nextNotificationDate }.

Required permissions
- "API Enabled" + "Schedule Reports".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_report_subscriptions({})
