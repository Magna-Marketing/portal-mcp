Tool: list_topics

Description
List `Topic` records (free-text tags applied to records). Returns name,
description, talking-about counts, and assignment counts.

Salesforce API
REST API (Data), v66.0.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, Name, Description, TalkingAbout, NetworkId, CreatedDate, SystemModstamp FROM Topic

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
namePattern (string, optional).
networkId (string, optional) - Restrict to one Experience site.
includeAssignmentCounts (boolean, optional, default true).
limit (number, optional, default 200).

Result
totalSize (number).
topics (array) - { id, name, description, talkingAbout, networkId, assignmentCount (when includeAssignmentCounts=true) }.

Required permissions
- "API Enabled".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_topics({})
