Tool: list_external_objects

Description
List External Objects (`__x`) and the data source each is bound to.
External Objects are queryable via SOQL but reads forward to the
external system.

Salesforce API
REST API + Tooling, v66.0. SOQL/EntityDefinition.

Endpoint
GET /services/data/v66.0/tooling/query/?q=SELECT QualifiedApiName, Label, PluralLabel, ExternalDataSourceId, ExternalDataSource.DeveloperName, ExternalName FROM EntityDefinition WHERE IsExternallyVisible = true AND ExternalDataSourceId != null

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
externalDataSourceName (string, optional).

Result
totalSize (number).
externalObjects (array) - { qualifiedApiName, label, pluralLabel, externalName, externalDataSourceId, externalDataSourceName }.

Required permissions
- "Customize Application".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_external_objects({})
