Tool: get_session_settings

Description
Read org-level session, password, and MFA settings: timeouts, lockout, password complexity, MFA requirement, identity confirmation, session security level, force re-login on IP change, etc. Use for security audits and reporting.

Salesforce API
Metadata API (`SecuritySettings`, `ProfilePasswordPolicy`, `ProfileSessionSetting`), v66.0.

Endpoint
POST /services/Soap/m/66.0   (readMetadata SecuritySettings + per-profile session/password settings)

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
includeProfileOverrides (boolean, optional, default true) - Include per-profile session and password policy overrides.

Result
generatedAt (string).
session (object):
  sessionTimeout (string) - "TwoHours" | "FourHours" | etc.
  forceLogoutOnSessionTimeout (boolean).
  forceRelogin (boolean).
  lockSessionsToDomain (boolean).
  lockSessionsToIp (boolean).
  requireHttpOnly (boolean).
  requireHttps (boolean).
  enableCSRFOnGet (boolean).
  redirectionWarning (boolean).
password (object):
  expiration (string) - "FortyFiveDays" | "Never" | etc.
  minimumLength (number).
  complexity (string) - "NoRestriction" | "AlphaNumeric" | "SpecialCharacters" | "UpperLowerCaseNumber" | "UpperLowerCaseNumberSpecialCharacters".
  historyRestriction (number) - Disallow last N passwords.
  maxLoginAttempts (number).
  lockoutInterval (string) - "Forever" | "FifteenMinutes" | "ThirtyMinutes" | "SixtyMinutes".
  questionRestriction (string).
mfa (object):
  enabledForAllUsers (boolean).
  identityConfirmationOnLogin (boolean).
  identityConfirmationOnReportExport (boolean).
profileOverrides (array, optional) - Per profile: { profileName, sessionTimeout, passwordExpiration, complexity, lockoutInterval }.

Required permissions
- "Manage Internal Users" / "View Setup and Configuration".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

get_session_settings({ "includeProfileOverrides": true })
