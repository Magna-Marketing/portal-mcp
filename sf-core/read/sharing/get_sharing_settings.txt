Tool: get_sharing_settings

Description
Get the org-wide defaults (OWD) for sharing on every relevant sObject: internal default, external default, grant access using hierarchies, and access level for related person accounts. This is the foundation of the sharing model.

Salesforce API
Metadata API (`SharingSettings` type) + Tooling API (`EntityDefinition`), v66.0.

Endpoint
POST /services/Soap/m/66.0    (readMetadata SharingSettings)
GET /services/data/v66.0/tooling/query/?q=SELECT QualifiedApiName, InternalSharingModel, ExternalSharingModel FROM EntityDefinition WHERE InternalSharingModel != null

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
sObjects (string[], optional) - Restrict to listed sObject API names. Omit for all.
includeStandardOnly (boolean, optional, default false).
includeCustomOnly (boolean, optional, default false).

Result
generatedAt (string).
settings (array) - Each:
  sObject (string).
  label (string).
  internalSharingModel (string) - "Private"|"Read"|"ReadWrite"|"ReadWriteTransfer"|"FullAccess"|"ControlledByParent"|"ControlledByLeadOrContact"|"ControlledByCampaign".
  externalSharingModel (string).
  grantAccessUsingHierarchies (boolean).
  isCustom (boolean).
externalSharingEnabled (boolean) - Org-level flag.

Required permissions
- "View Setup and Configuration".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

get_sharing_settings({ "sObjects": ["Account", "Opportunity", "Project__c"] })
