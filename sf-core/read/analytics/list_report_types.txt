Tool: list_report_types

Description
List report types available in the org with their labels, base sObject,
and category info. A report type defines what fields and joins a report
can use; selecting the right one is required when creating a report
(`create_report`).

Salesforce API
Analytics REST API, v66.0. `GET /analytics/reportTypes`.

Endpoint
GET /services/data/v66.0/analytics/reportTypes

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
type (string, optional) - "standard" | "custom" | "all" (default "all").
includeDeprecated (boolean, optional, default false).

Result
reportTypes (array) - Each:
  type (string) - Developer name (e.g. "AccountList", "Custom_Project_With_Tasks__c").
  label (string).
  category (string) - "accounts", "opportunities", "custom", etc.
  isStandard (boolean).
  isDeprecated (boolean).
  describeUrl (string) - For drilling in.

Required permissions
- "API Enabled".

Idempotency & limits
- Read-only; safe to retry.
- Result set is generally small (<500); single page.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_report_types({ "type": "custom" })

Example result

{
  "reportTypes": [
    { "type": "Custom_Project_With_Tasks__c", "label": "Project With Tasks", "category": "custom", "isStandard": false, "isDeprecated": false }
  ]
}
