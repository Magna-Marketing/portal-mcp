Tool: list_licenses

Description
List the org's user licenses, permission set licenses, and (optionally) feature licenses. Returns total / used / available counts and the assignments for each. Use to confirm capacity before bulk user creation, permission set rollout, or feature enablement.

Salesforce API
REST API (Data), v66.0. SOQL on `UserLicense`, `PermissionSetLicense`, `PermissionSetLicenseAssign`, `UserPackageLicense` (managed package), `FeatureLicense`.

Endpoint
GET /services/data/v66.0/query/?q=SELECT Id, Name, MasterLabel, Status, TotalLicenses, UsedLicenses, MonthlyLoginsEntitlement, MonthlyLoginsUsed FROM UserLicense
GET /services/data/v66.0/query/?q=SELECT Id, MasterLabel, DeveloperName, TotalLicenses, UsedLicenses, ExpirationDate, Status, MaximumPermissionsActivities, IsAvailableForIntegrations FROM PermissionSetLicense
GET /services/data/v66.0/query/?q=SELECT PermissionSetLicenseId, AssigneeId, Assignee.Username FROM PermissionSetLicenseAssign WHERE PermissionSetLicenseId IN (...)
GET /services/data/v66.0/query/?q=SELECT Id, FeatureType, Name, TotalCount, UsedCount FROM FeatureLicense

Headers
Authorization: Bearer <access_token>
Accept: application/json

Body
None.

Input (parameters)
type (string, optional, default "all") - "user" | "permset" | "package" | "feature" | "all".
includeAssignments (boolean, optional, default false) - For permset licenses, include the assignee list (heavier).
nearExhaustionThreshold (number, optional, default 0.9) - 0..1; flag entries where usedLicenses / totalLicenses ≥ threshold.

Result
generatedAt (string).
userLicenses (array) - { id, name, masterLabel, status, totalLicenses, usedLicenses, availableLicenses, monthlyLoginsEntitlement, monthlyLoginsUsed }.
permissionSetLicenses (array) - { id, masterLabel, developerName, totalLicenses, usedLicenses, availableLicenses, expirationDate, status, assignments (optional) }.
packageLicenses (array, optional).
featureLicenses (array, optional) - { name, type, totalCount, usedCount, availableCount }.
warnings (array) - Entries near or at exhaustion.

Required permissions
- "Manage Internal Users" / "View Setup and Configuration".

Idempotency & limits
- Read-only.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY.

Example request

list_licenses({ "type": "all", "includeAssignments": false })
