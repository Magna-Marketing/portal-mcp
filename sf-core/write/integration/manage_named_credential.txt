Tool: manage_named_credential

Description
Create / update / delete Named Credentials. Stores endpoint URL plus
auth (BasicAuth, OAuth, AWS Sig V4, Custom). Pair with
`manage_external_credential` for principal-mapped OAuth flows.

Salesforce API
Connect API + Metadata API, v66.0.

Endpoint
POST   /services/data/v66.0/connect/named-credentials/named-credentials
PATCH  /services/data/v66.0/connect/named-credentials/named-credentials/{developerName}
DELETE /services/data/v66.0/connect/named-credentials/named-credentials/{developerName}

Headers
Authorization: Bearer <access_token>
Content-Type: application/json

Body
{
  "developerName": "MTG_External_API",
  "masterLabel": "MTG External API",
  "type": "SecuredEndpoint",
  "endpoint": "https://api.partner.com",
  "principalType": "NamedUser",
  "authProtocol": "OAuth",
  "externalCredential": "MTG_Partner_OAuth",
  "generateAuthorizationHeader": true,
  "allowMergeFieldsInHeader": true,
  "allowMergeFieldsInBody": true,
  "calloutOptionAllowSecretInPayload": false,
  "calloutSettings": { "outboundNetworkConnection": null }
}

Input (parameters)
operation (string, required) - "create" | "update" | "delete".
developerName (string, required).
masterLabel (string, required for create).
type (string, optional, default "SecuredEndpoint") - "SecuredEndpoint" | "Endpoint".
endpoint (string, required for create).
principalType (string, optional) - "Anonymous" | "NamedUser" | "PerUser".
authProtocol (string, optional) - "NoAuthentication" | "Password" | "OAuth" | "AwsSv4" | "Jwt" | "Custom".
externalCredentialName (string, optional) - For OAuth/External Credential pairing.
generateAuthorizationHeader (boolean, optional, default true).
allowMergeFieldsInHeader (boolean, optional, default true).
allowMergeFieldsInBody (boolean, optional, default false).
calloutOptionAllowSecretInPayload (boolean, optional, default false).
calloutSettings (object, optional) - { outboundNetworkConnection }.

Result
namedCredential (object) - { id, developerName, masterLabel, endpoint, principalType, authProtocol, externalCredentialName }.

Required permissions
- "Customize Application" + "Manage External Connections".

Idempotency & limits
- update idempotent.

Errors
- DUPLICATE_DEVELOPER_NAME, INVALID_FIELD.

Example request

manage_named_credential({
  "operation": "create",
  "developerName": "MTG_Partner_API",
  "masterLabel": "MTG Partner API",
  "endpoint": "https://api.partner.com",
  "principalType": "PerUser",
  "authProtocol": "OAuth",
  "externalCredentialName": "MTG_Partner_OAuth"
})
