Tool: update_mydomain_settings

Description
Update MyDomain configuration: login policy, redirect policy, embedded
login disabled, partial login policy. Domain rename is a heavy operation
(propagates to login URLs) and not exposed by this tool.

Salesforce API
Metadata API (`MyDomainSettings`), v66.0.

Endpoint
POST /services/data/v66.0/metadata/deployRequest

Headers
Authorization: Bearer <access_token>
Content-Type: application/json

Body
N/A.

Input (parameters)
loginPolicy (string, optional) - "All" | "MyDomainOnly".
redirectPolicy (string, optional) - "Disabled" | "Warning" | "Frame" | "Redirect".
disableEmbeddedLogin (boolean, optional).
partialLoginPolicy (string, optional).
useNativeBrowserForLogin (boolean, optional).
checkOnly (boolean, optional, default false).

Result
deploy (object).
settings (object) - Updated values.

Required permissions
- "Customize Application".

Idempotency & limits
- Idempotent on equal input.

Errors
- INSUFFICIENT_ACCESS_OR_READONLY, INVALID_FIELD.

Example request

update_mydomain_settings({ "loginPolicy": "MyDomainOnly", "redirectPolicy": "Disabled" })
